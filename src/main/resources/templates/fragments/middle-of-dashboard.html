Welcome To Projectory

Getting Started:

For Project Owners:
Start by adding a project by clicking the button top left.

Hereafter you can assign people to your project including project managers and team members. If you have any sub-projects, tasks and subtasks already planned you can add those by navigating through the sub-project, task and sub-task sections.

Click the statistics button in the navigation bar to view statistics of the selected project.

For Project Managers
Assuming you're assigned to a project you have the options to add sub-projects, tasks and subtasks to the current project.

For Developers:
Assuming you've been assigned to one or more projects you can navigate to those in the side panel. Here you can check uncompleted projects, sub-projects, tasks and subtask.
As you're working on a project you can input your work-hours on the selected projects you've been assigned to.
